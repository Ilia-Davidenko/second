## Списки

Чтобы добавить ненумерованные списки необходимо пункты выделить звездочкой, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить пронумерованные списки, необходимо их просто пронумеровать, вот так:
1. Первый пункт
2. Второй пункт
